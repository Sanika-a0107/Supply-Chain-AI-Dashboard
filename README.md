# Supply Chain Delay Prediction & Risk Dashboard

## 📌 Project Overview
This project identifies the root causes of shipping delays in a global supply chain using SQL, Python (AI), and Power BI.

## 🛠️ Tech Stack
- **Database:** PostgreSQL (Data Cleaning & Extraction)
- **AI/ML:** Python (Scikit-Learn, Random Forest Regressor)
- **Visualization:** Matplotlib & Power BI
- **Environment:** VS Code

## 🚀 Key Insights
- **Root Cause:** AI Model identified that **Days Scheduled** (95.75% importance) is the primary driver of delays, suggesting unrealistic logistics planning.
- **KPI:** The current **Late Delivery Rate is ~54.8%**, visualized through an interactive Power BI dashboard.

## 📁 Files
- `visualize_results.py`: Generates the feature importance chart.
- `late_delivery_analysis.png`: Visual output of the AI model.
- `Supply_Chain_Dashboard.pbix`: Interactive dashboard for regional analysis.
